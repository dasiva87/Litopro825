# 1. ¿Qué es GrafiRed?

**GrafiRed** es una plataforma SaaS (Software como Servicio) diseñada específicamente para **litografías, imprentas y empresas de artes gráficas**. Es una solución integral que permite gestionar todo el ciclo de vida de un trabajo de impresión, desde la cotización inicial hasta el cobro final.

## Características principales

- **Gestión completa de documentos**: Cotizaciones, órdenes de pedido, órdenes de producción y cuentas de cobro
- **Cálculos automáticos de impresión**: El sistema calcula automáticamente costos de papel, impresión, corte y acabados
- **Multi-empresa**: Cada empresa tiene su espacio privado y seguro
- **Red de proveedores**: Conecta con otros litógrafos para tercerizar trabajos
- **Gestión de inventario**: Control de papeles, productos y máquinas
- **Proyectos**: Agrupa múltiples trabajos relacionados en un solo proyecto

## ¿Para quién es GrafiRed?

- Litografías pequeñas y medianas
- Imprentas digitales y offset
- Empresas de artes gráficas
- Freelancers del sector gráfico
- Proveedores de servicios de impresión

# 6. Módulos del sistema

## 6.1 Documentos

### Cotizaciones
El punto de partida de todo trabajo. Una cotización incluye:
- Cliente
- Fecha y validez
- Items (trabajos a realizar)
- Descuentos e IVA
- Total

**Estados de una cotización:**
```
BORRADOR → ENVIADA → EN PROCESO → COMPLETADA
                  ↘ CANCELADA
```

### Órdenes de Pedido
Cuando necesitas comprar materiales o tercerizar trabajo:
- Se generan automáticamente desde cotizaciones
- Agrupa items por proveedor
- Rastrea confirmaciones y entregas

**Estados:**
```
BORRADOR → ENVIADA → CONFIRMADA → RECIBIDA PARCIALMENTE → COMPLETADA
                                ↘ CANCELADA
```

### Órdenes de Producción
Para controlar el trabajo en tu taller:
- Asigna operadores
- Registra tiempos
- Control de calidad

**Estados:**
```
BORRADOR → EN PROCESO → EN ESPERA → COMPLETADA
                      ↘ CANCELADA
```

### Cuentas de Cobro
Para facturar al cliente:
- Se genera desde órdenes de producción
- Incluye todos los items realizados
- Rastrea pagos

**Estados:**
```
BORRADOR → ENVIADA → APROBADA → PAGADA
                   ↘ CANCELADA
```

## 6.2 Tipos de Items

GrafiRed soporta diferentes tipos de trabajos:

### Item Sencillo (SimpleItem)
Para trabajos básicos de impresión:
- Volantes, tarjetas, afiches
- Papelería corporativa
- Cualquier impresión plana

**Campos principales:**
- Tamaño del trabajo (ancho × alto)
- Papel a usar
- Máquina de impresión
- Tintas (frente y reverso)
- Cantidad

### Revista/Catálogo (MagazineItem)
Para publicaciones con múltiples páginas:
- Revistas
- Catálogos
- Libros

**Campos principales:**
- Tamaño cerrado
- Número de páginas
- Tipo de encuadernación
- Papel de portada y páginas internas

### Talonario (TalonarioItem)
Para libretitas y talonarios numerados:
- Facturas
- Recibos
- Comandas

**Campos principales:**
- Número inicial y final
- Cantidad de talonarios
- Hojas por talonario

### Item Digital (DigitalItem)
Para servicios que no son de impresión:
- Diseño gráfico
- Fotografía
- Servicios digitales

### Producto (Product)
Para productos de inventario:
- Productos terminados
- Insumos
- Mercancía

### Item Personalizado (CustomItem)
Para cualquier otro servicio:
- Servicios especiales
- Mano de obra
- Extras

## 6.3 Proyectos

Los proyectos permiten agrupar múltiples documentos relacionados:

```
PROYECTO: "Campaña Publicitaria Cliente X"
├── Cotización #0045 - Volantes 10.000 unidades
├── Cotización #0046 - Tarjetas de presentación
├── Cotización #0047 - Pendones
├── Orden de Pedido #0012 - Proveedor de papel
├── Orden de Producción #0008 - Impresión volantes
└── Cuenta de Cobro #0003 - Factura final
```

**Beneficios:**
- Visión integral del proyecto
- Presupuesto vs. real
- Fechas de inicio y entrega
- Estado general del proyecto

# 7. Tutorial: Flujo de trabajo completo

## Escenario
Un cliente te pide cotizar **5.000 volantes tamaño carta a full color**.

---

## Paso 1: Crear la cotización

### 1.1 Ir a Cotizaciones
1. En el menú lateral, haz clic en **Documentos**
2. Selecciona **Cotizaciones**
3. Haz clic en el botón **+ Crear Cotización**

### 1.2 Datos básicos
1. **Cliente**: Busca el cliente o créalo con el botón "+"
2. **Fecha**: Se pone automáticamente (hoy)
3. **Válida hasta**: Selecciona hasta cuándo es válida la cotización
4. **Proyecto** (opcional): Asigna a un proyecto existente

### 1.3 Guardar borrador
1. Haz clic en **Crear**
2. El sistema genera el número automáticamente (ej: 0045)

---

## Paso 2: Agregar el item (volantes)

### 2.1 Ir a la pestaña Items
1. En la cotización recién creada, busca la sección **Items**
2. Haz clic en **+ Agregar Item Sencillo**

### 2.2 Configurar el trabajo

**Información básica:**
```
Descripción: Volantes full color
Cantidad: 5000
```

**Tamaño del trabajo:**
```
Ancho: 21.5 cm (carta)
Alto: 28 cm (carta)
```

**Papel:**
```
Papel: Propalcote 115gr (selecciona de la lista)
```

**Máquina:**
```
Máquina: Offset 4 colores (selecciona de la lista)
```

**Tintas:**
```
Tintas frente: 4 (CMYK - full color)
Tintas reverso: 4 (CMYK - full color)
```

### 2.3 Cálculo automático

El sistema calcula automáticamente:

```
┌─────────────────────────────────────────────┐
│  RESUMEN DE CÁLCULOS                        │
├─────────────────────────────────────────────┤
│  Tamaño del pliego: 70 × 100 cm             │
│  Tamaño de la hoja: 50 × 35 cm (máquina)    │
│  Montaje: 4 volantes por hoja               │
│  Hojas necesarias: 1,250                    │
│  Pliegos necesarios: 313                    │
├─────────────────────────────────────────────┤
│  Costo papel:        $156,500               │
│  Costo impresión:    $125,000               │
│  Costo corte:        $15,000                │
│  Margen (30%):       $88,950                │
├─────────────────────────────────────────────┤
│  PRECIO FINAL:       $385,450               │
└─────────────────────────────────────────────┘
```

### 2.4 Agregar acabados (opcional)

Si el volante lleva acabado especial:
1. En la sección **Acabados**, haz clic en **+ Agregar**
2. Selecciona el acabado (ej: "Laminado mate")
3. El sistema suma el costo al precio final

### 2.5 Guardar el item
1. Haz clic en **Guardar**
2. El item aparece en la lista de la cotización

---

## Paso 3: Revisar y ajustar la cotización

### 3.1 Resumen financiero

En la cotización verás el resumen:

```
┌─────────────────────────────────────────────┐
│  RESUMEN FINANCIERO                         │
├──────────────┬──────────────────────────────┤
│  Subtotal    │  $385,450                    │
│  Descuento   │  Sin descuento               │
│  IVA (19%)   │  Sin IVA                     │
│  TOTAL       │  $385,450                    │
└──────────────┴──────────────────────────────┘
```

### 3.2 Aplicar descuento (opcional)
1. En el campo **Descuento**, ingresa el porcentaje (ej: 10%)
2. El sistema recalcula automáticamente

### 3.3 Aplicar IVA (opcional)
1. Activa el toggle **Aplicar IVA**
2. El IVA por defecto es 19%
3. El sistema recalcula el total

### 3.4 Resultado final

```
┌─────────────────────────────────────────────┐
│  RESUMEN FINANCIERO                         │
├──────────────┬──────────────────────────────┤
│  Subtotal    │  $385,450                    │
│  Descuento   │  -$38,545 (10%)              │
│  IVA (19%)   │  +$65,912                    │
│  TOTAL       │  $412,817                    │
└──────────────┴──────────────────────────────┘
```

---

## Paso 4: Enviar cotización al cliente

### 4.1 Generar PDF
1. Haz clic en el botón **Imprimir PDF**
2. Se abre una nueva pestaña con el PDF profesional
3. Puedes descargarlo o imprimirlo

### 4.2 Enviar por email
1. Haz clic en **Enviar Email al Cliente**
2. Confirma el envío
3. El sistema envía el PDF al email del cliente
4. El estado cambia a **ENVIADA**

---

## Paso 5: Cliente aprueba → Crear Orden de Pedido

Cuando el cliente aprueba la cotización:

### 5.1 Cambiar estado
1. Edita la cotización
2. Cambia el estado a **En Proceso**
3. Guarda

### 5.2 Crear Orden de Pedido
1. En la cotización, haz clic en **Crear Orden de Pedido**
2. El sistema crea automáticamente la orden
3. **Si hay items de diferentes proveedores**, crea una orden por cada proveedor

### 5.3 Enviar al proveedor
1. Abre la Orden de Pedido
2. Revisa los items y cantidades
3. Haz clic en **Enviar al Proveedor**
4. El proveedor recibe notificación (si está en GrafiRed)

---

## Paso 6: Crear Orden de Producción

Cuando tienes los materiales:

### 6.1 Desde la cotización
1. Abre la cotización
2. Haz clic en **Crear Orden de Producción**
3. Se crea la orden con todos los items

### 6.2 Asignar operador
1. Abre la Orden de Producción
2. Asigna el operador responsable
3. Agrega notas de producción si es necesario

### 6.3 Iniciar producción
1. Haz clic en **Iniciar Producción**
2. El estado cambia a **En Proceso**
3. Se registra la fecha y hora de inicio

### 6.4 Seguimiento
Durante la producción puedes:
- Pausar el trabajo (**En Espera**)
- Agregar notas de avance
- Registrar problemas

### 6.5 Finalizar producción
1. Cuando termines, haz clic en **Completar**
2. El estado cambia a **Completada**
3. Se registra la fecha de finalización

---

## Paso 7: Crear Cuenta de Cobro

Trabajo terminado, hora de cobrar:

### 7.1 Desde la Orden de Producción
1. Abre la orden completada
2. Haz clic en **Crear Cuenta de Cobro**
3. Se genera con todos los items y precios

### 7.2 Revisar la cuenta
1. Verifica los items y totales
2. Ajusta si es necesario
3. El estado es **Borrador**

### 7.3 Enviar al cliente
1. Haz clic en **Enviar al Cliente**
2. Se envía el PDF por email
3. El estado cambia a **Enviada**

### 7.4 Cliente aprueba
1. Cuando el cliente aprueba, cambia a **Aprobada**
2. Esto indica que está listo para pagar

### 7.5 Registrar pago
1. Cuando recibas el pago, cambia a **Pagada**
2. El flujo está completo

---

## Resumen del flujo

```
COTIZACIÓN (Borrador)
    ↓ [Enviar al cliente]
COTIZACIÓN (Enviada)
    ↓ [Cliente aprueba]
COTIZACIÓN (En Proceso)
    ↓ [Crear Orden de Pedido]
ORDEN DE PEDIDO → Proveedor
    ↓ [Recibir materiales]
    ↓ [Crear Orden de Producción]
ORDEN DE PRODUCCIÓN (En Proceso)
    ↓ [Completar trabajo]
ORDEN DE PRODUCCIÓN (Completada)
    ↓ [Crear Cuenta de Cobro]
CUENTA DE COBRO (Enviada)
    ↓ [Cliente paga]
CUENTA DE COBRO (Pagada) ✓
```

# 8. Gestión de Proyectos

## ¿Qué es un proyecto?

Un proyecto agrupa múltiples documentos relacionados con un mismo cliente o campaña.

## ¿Cuándo usar proyectos?

- Campañas publicitarias con múltiples piezas
- Clientes recurrentes con varios trabajos
- Proyectos grandes con entregas parciales
- Licitaciones con múltiples ítems

## Crear un proyecto

### Paso 1: Ir a Proyectos
1. Menú **Documentos** → **Proyectos**
2. Clic en **+ Crear Proyecto**

### Paso 2: Datos del proyecto
```
Nombre: Campaña Navidad 2026 - Cliente X
Descripción: Material publicitario para temporada navideña
Estado: Activo
Contacto: Cliente X
Fecha inicio: 15/11/2026
Fecha estimada fin: 15/12/2026
Presupuesto: $5,000,000
```

### Paso 3: Crear cotizaciones dentro del proyecto
1. Abre el proyecto
2. En la pestaña **Cotizaciones**, haz clic en **+ Nueva Cotización**
3. La cotización se vincula automáticamente al proyecto

### Paso 4: Ver resumen del proyecto

El proyecto muestra:
```
┌─────────────────────────────────────────────┐
│  PROYECTO: Campaña Navidad 2026             │
├─────────────────────────────────────────────┤
│  Estado: En Proceso                         │
│  Presupuesto: $5,000,000                    │
│  Total cotizado: $4,250,000                 │
│  Total cobrado: $2,100,000                  │
├─────────────────────────────────────────────┤
│  DOCUMENTOS                                 │
│  • 3 Cotizaciones                           │
│  • 2 Órdenes de Pedido                      │
│  • 2 Órdenes de Producción                  │
│  • 1 Cuenta de Cobro                        │
└─────────────────────────────────────────────┘
```

## Estados de un proyecto

```
BORRADOR → ACTIVO → EN PROCESO → COMPLETADO
              ↓          ↓
          EN ESPERA  CANCELADO
```

# 9. Gestión de Inventario

## 9.1 Papeles

### Ver papeles disponibles
1. Menú **Inventario** → **Papeles**
2. Verás la lista con stock actual

### Crear un papel
1. Clic en **+ Crear Papel**
2. Completa:
```
Código: PPC-115
Nombre: Propalcote 115gr
Gramaje: 115 gr/m²
Ancho: 70 cm
Alto: 100 cm
Costo por pliego: $500
Precio de venta: $650
Stock actual: 500 pliegos
Stock mínimo: 100 pliegos
Proveedor: Papelería XYZ
```

### Alertas de stock
Cuando el stock baja del mínimo:
- Aparece alerta en el dashboard
- El papel se marca en rojo en la lista

## 9.2 Máquinas de impresión

### Crear una máquina
1. Menú **Inventario** → **Máquinas**
2. Clic en **+ Crear Máquina**
3. Completa:
```
Nombre: Offset Heidelberg 4 colores
Tipo: Offset
Ancho máximo: 50 cm
Alto máximo: 35 cm
Colores máximos: 4
Costo por impresión: $100
Costo CTP: $25,000
```

### ¿Por qué es importante?
- El sistema usa las dimensiones para calcular el montaje
- El costo por impresión se aplica automáticamente
- Puedes tener máquinas propias y de proveedores

## 9.3 Acabados

### Crear un acabado
1. Menú **Inventario** → **Acabados**
2. Clic en **+ Crear Acabado**
3. Completa:
```
Nombre: Laminado mate
Descripción: Acabado mate protector
Precio por unidad: $50
Unidad de medida: cm²
Proveedor: Acabados Express (o "Es propio")
```

### Tipos de acabados comunes
- Laminado (mate, brillante)
- Barniz UV
- Troquelado
- Plegado
- Encuadernación
- Hot stamping

# 10. Gestión de Contactos

## 10.1 Tipos de contactos

| Tipo | Descripción |
|------|-------------|

| **Proveedor** | Quien te vende materiales o servicios |

1. Menú **Contactos** → **Clientes**

```
Email: juan@empresaabc.com
Teléfono: 3001234567
Dirección: Calle 123 #45-67

Límite de crédito: $2,000,000
Plazo de pago: 30 días
Descuento base: 5%
```

1. Menú **Contactos** → **Proveedores**
2. Clic en **+ Crear Proveedor**
3. Completa similar al cliente

Si tu proveedor también usa GrafiRed:
1. En el campo **Empresa vinculada**, busca su empresa
2. Al vincular, podrás:
   - Enviar órdenes de pedido directamente
   - Ver estados en tiempo real
   - Compartir información de productos

# 12. Preguntas frecuentes

## General

**¿Puedo usar GrafiRed en mi celular?**
Sí, GrafiRed es responsive y funciona en cualquier dispositivo con navegador.

**¿Mis datos están seguros?**
Sí, usamos encriptación SSL y respaldos diarios automáticos.

**¿Puedo exportar mis datos?**
Sí, puedes exportar cotizaciones, clientes y reportes en PDF y Excel.

## Cotizaciones

**¿Cómo cambio el margen de ganancia?**
En cada item puedes ajustar el porcentaje de margen. También puedes configurar un margen predeterminado en la configuración.

**¿Puedo duplicar una cotización?**
Sí, abre la cotización y usa la opción "Nueva Versión" para crear una copia.

**¿Cómo agrego un item que no es de impresión?**
Usa "Item Personalizado" o "Item Digital" para servicios que no requieren cálculos de impresión.

## Inventario

**¿Cómo actualizo el stock de papel?**
El stock se actualiza automáticamente cuando creas órdenes de pedido y producción. También puedes ajustarlo manualmente.

**¿Puedo tener papeles de varios proveedores?**
Sí, cada papel puede tener un proveedor diferente asignado.

## Proveedores

**¿Cómo tercerizo un trabajo?**
Al crear la orden de pedido, el sistema detecta si el item requiere un proveedor externo y te permite seleccionarlo.

**¿El proveedor ve mi precio de venta?**
No, el proveedor solo ve el precio de compra acordado, no tu precio de venta al cliente final.

## Proyectos

**¿Puedo mover una cotización a otro proyecto?**
Sí, edita la cotización y cambia el proyecto asignado.

**¿Qué pasa si no asigno proyecto?**
Las cotizaciones sin proyecto funcionan normalmente, solo que no están agrupadas.
